# Provided apps in the Python Code folder
## Clock
Displays an analogue clock. Requires a wifi connection to get the correct time

## FlappyBird
A flappybird game clone

## Happy
Using the button make the esp32 smile or be sad

## Light
Using the button, turn the LED on or off

## Logo
Display the Scout Logo

## Main Menu
Display a menu allowing access to all the python apps on the ESP32-C3

## Message Simple
Simple bluetooth messaging app, allows user to send or receive from a preset list of phrases

## Message
More complicated messaging app. User can set a username, and use a keyboard to send a message to all devices in range or just a specific user

## Pet
A tamagotchi clone, keep the pet fed and happy

## Simple_ESP
A library of useful python classes to make programming the device easier

## Tetris
A simple side-ways tetris clone

## Weather
Display the current weather in Bury St Edmunds. Requires wifi connection

## Wifi
Connect to the wifi. Will save the wifi connection details.
To avoid having to type the password in the microcontroller, add your info to registry.json file on the device
{"wifi.password": "your_password", "wifi.ssid": "your_ssid"}
